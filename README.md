# Portafolio profesional

Sitio web personal de **Gustavo Abel Caceres Martinez**, orientado a presentar su perfil profesional, experiencia, areas de especializacion y medios de contacto.

## Descripcion

Este proyecto es un portafolio web estatico desarrollado con HTML, CSS y JavaScript. Su contenido principal incluye presentacion profesional, trayectoria, galeria, mapa interactivo y seccion de contacto.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Leaflet

## Estructura basica

```text
.
|-- index.html
|-- styles.css
|-- script.js
|-- assets/
|-- data/
`-- README.md
```

## Uso

Abre `index.html` en tu navegador para visualizar el sitio localmente.
